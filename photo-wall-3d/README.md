# 🌌 3D 星海照片墙

> 白鹿 · 环绕式星海银河体验

## ✨ 特色功能

- **3D 圆柱环绕** — 图片绕成一圈，点击/拖拽旋转
- **星空粒子背景** — 动态流星+星云，沉浸感满分
- **自动轮播** — 可调速：慢/中/快
- **分类筛选** — 红毯/写真/剧照/日常/综艺/广告
- **缩略图条** — 快速定位任意图片
- **图片放大** — 点击中心图片全屏查看
- **键盘控制** — ←→翻页 空格暂停 ↑↓切换分类
- **触控支持** — 手机端滑动翻页
- **精美装饰框** — 金色相框围绕中心图片

## 📂 文件说明

- `index.html` — 主页面（独立运行，无需服务器）
- `preview.html` — GitHub Pages 预览引导页
- `config.json` — 数据源配置（从 ai-beauty-gallery 读取）

## 🔧 使用方法

直接在浏览器打开 `index.html` 即可运行。

## 📡 数据来源

自动从 [ai-beauty-gallery](https://github.com/qianjin1111/ai-beauty-gallery) 读取图片列表。
