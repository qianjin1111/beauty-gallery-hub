# 🌸 AI Beauty Gallery

> AI生成的美女图片相册

## 📖 项目简介

本项目是一个私人相册，用于存储和展示AI生成的美女图片。

## 🎨 图片内容

- AI生成的美女图片
- 风格多样化（写实、二次元、艺术等）
- 定期更新

## 📁 项目结构

```
ai-beauty-gallery/
├── README.md              # 项目说明
├── images/               # 图片目录
│   ├── realistic/        # 写实风格
│   ├── anime/            # 二次元风格
│   ├── artistic/         # 艺术风格
│   └── others/           # 其他风格
└── .gitignore            # Git忽略配置
```

## 🚀 使用方法

### 添加图片

1. 将图片放到对应的目录
2. 提交到GitHub：
   ```bash
   git add .
   git commit -m "添加新图片"
   git push origin main
   ```

### 查看图片

1. 访问GitHub仓库
2. 浏览图片目录
3. 点击图片查看
