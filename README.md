# 🌸 Beauty Gallery Hub

> 美女图片中心 - AI生成 + 白鹿照片墙

## 📖 项目简介

本项目是一个综合性的美女图片中心，包含：

1. **AI生成图片**：使用AI技术生成的多样化美女图片
2. **白鹿照片墙**：白鹿明星的照片展示墙，支持轮播和管理

## 📖 使用方法

### 使用白鹿照片墙

1. 访问照片墙主页
2. 查看自动轮播的白鹿照片
3. 使用控制按钮进行操作
4. 在"添加图片"区域添加新照片
5. 点击"刷新图片"获取更多照片

详细说明请查看：[photo-wall/README.md](./photo-wall/README.md)

### 添加AI图片

1. 将图片按风格分类放入对应目录
2. 提交到 GitHub：
   ```bash
   git add images/
   git commit -m "添加新图片"
   git push origin main
   ```

### 查看图片

1. 访问 GitHub 仓库
2. 浏览对应明星目录
3. 点击图片查看大图
