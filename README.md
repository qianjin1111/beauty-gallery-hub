## 🚀 使用方法

### 添加图片

1. 将图片按风格分类放入对应目录
2. 提交到 GitHub：
   ```bash
   git add .
   git commit -m "添加新图片"
   git push origin main
   ```

### 查看图片

1. 访问 GitHub 仓库
2. 浏览对应明星目录
3. 点击图片查看大图
