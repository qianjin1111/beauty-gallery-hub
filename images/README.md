# 📷 图片说明

## 📂 目录说明

### `realistic/` - 写实风格
- 真实感强的美女图片
- 接近摄影照片的效果
- 适合作为头像、壁纸

### `anime/` - 二次元风格
- 动漫、漫画风格的美女图片
- 包含日系、韩系等风格
- 适合ACG爱好者

### `artistic/` - 艺术风格
- 艺术、插画风格的美女图片
- 包含油画、水彩、素描等风格
- 适合艺术欣赏

### `others/` - 其他风格
- 其他风格的美女图片
- 混合风格或特殊风格
- 不属于上述分类的图片

## 🏷️ 图片命名规范

建议使用以下命名格式：
- `YYYY-MM-DD-风格描述-序号.jpg`
- 例如：`2024-04-15-realistic-portrait-001.jpg`

示例：
- `2024-04-15-realistic-portrait-001.jpg`
- `2024-04-15-anime-girl-001.jpg`
- `2024-04-15-artistic-oil-painting-001.jpg`

## 📝 图片元数据

建议在图片文件名中包含以下信息：
- 日期（YYYY-MM-DD）
- 风格（realistic/anime/artistic/others）
- 主题描述（portrait/cosplay/scenery等）
- 序号（001, 002, 003...）

## 🔄 定期整理

建议：
- 每月整理一次图片
- 删除质量不佳的图片
- 合并重复的图片
- 优化目录结构
