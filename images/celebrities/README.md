# 📷 明星照片分类指南

> 为白鹿和其他明星的照片建立标准化的分类体系

---

## 🌟 明星照片分类体系

### 一级分类：celebrities（明星专区）

所有明星照片都放在 `images/celebrities/` 目录下。

### 二级分类：明星名称

每个明星一个独立的文件夹，使用拼音或英文命名。

```
images/celebrities/
├── bai-lu/          # 白鹿
├── yang-zi/         # 杨紫
├── dilraba/         # 迪丽热巴
├── zhao-liying/     # 赵丽颖
└── others/          # 其他明星
```

### 三级分类：照片场景

每个明星下按照场景进行分类。

## 🏷️ 通用命名规范

### 格式

```
YYYY-MM-DD-分类-描述-序号.扩展名
```

### 各部分说明

| 部分 | 说明 | 示例 |
|------|------|------|
| YYYY-MM-DD | 日期 | 2024-04-15 |
| 分类 | 照片分类 | red-carpet, photoshoot, tv-series等 |
| 描述 | 活动名称/作品名/品牌名 | 时尚芭莎年度盛典, 周生如故, 欧莱雅 |
| 序号 | 当天照片的序号 | 001, 002, 003 |
| 扩展名 | 文件格式 | jpg, png, webp |

### 特殊说明

**日期**：
- 如果照片日期不确定，使用 `XXXX-XX-XX`
- 如果知道大概时间，使用近似日期

**序号**：
- 建议使用3位数字：001, 002, 003
- 如果照片很多，可以使用4位：0001, 0002

**描述**：
- 尽量简洁明了
- 使用中英文都可以
- 避免使用特殊字符

## 🔄 维护建议

### 定期整理

- **每周**：添加新照片
- **每月**：整理和分类
- **每季度**：全面整理和优化

### 质量控制

- 删除重复照片
- 删除模糊/低质量照片
- 优化照片分类
- 统一命名规范

### 备份建议

- 定期备份到其他平台
- 考虑使用云存储
- 注意版权问题
