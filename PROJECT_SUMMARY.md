## 🚀 如何使用

### 方式1：在GitHub上直接操作

1. 访问：https://github.com/qianjin1111/ai-beauty-gallery
2. 点击"Upload files"
3. 选择图片上传到对应目录
4. 提交描述并上传

### 方式2：使用Git命令行

```bash
# 克隆仓库（可选）
git clone git@github.com:qianjin1111/ai-beauty-gallery.git
cd ai-beauty-gallery

# 添加图片
# 将图片放到 images/realistic/ 目录

# 提交到GitHub
git add .
git commit -m "添加新图片"
git push origin main
```

如果图片文件很大（>10MB），建议：
```bash

git lfs track "*.png"
```
